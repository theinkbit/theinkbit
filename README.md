## Hey, I'm Blake

I'm 17, from Walsall in the West Midlands. I've been into tech for about five years now. It started with me watching YouTube tutorials and somehow I ended up running servers and writing APIs. Bit of a rabbit hole.

I've done sys admin stuff for a few hosting companies and ran my own host for a while before it got shut down. Learnt loads from it though. These days I mostly build things around music streaming and file hosting, and spend way too long at 3am fixing configs that were working fine before I touched them.

I've got a Dell PowerEdge R820 at home that runs most of my virtualisation stuff. Proxmox with KVM/QEMU, messed around with GPU passthrough and hiding VMs from anti-cheat with SMBIOS and Hyper-V tweaks. I've also had a couple RPi4s and an RPi5 for random projects.

I prefer backend over frontend. I can make stuff look alright if I have to but I'd much rather be making sure the server actually stays up than picking colours for buttons.

### What I've Built

**[SYNK Music](https://synkmusic.com)** - Formerly Encore. Built on top of [Swing Music](https://github.com/swingmx/swingmusic), turning it into a remote streaming server so you don't need files on your device. **[Mimso Blog](https://blog.mimso.co.uk)** - Where I post updates and that. Built it with Express cos I wanted full control over it.
